## 📂 Projects

### 1. 💳 Credit Card Default Prediction
**File:** `Credit_Card_Default_Prediction.ipynb`

Engineered a binary classifier to identify high-risk customers from a dataset of 30,000 records with a 78-22 class imbalance.

| Metric | Value |
|---|---|
| AUC Score | **0.781** |
| Models Used | Logistic Regression, Decision Tree, XGBoost |
| Imbalance Handling | Class Weighting |
| Key Predictor | Payment History (35% importance) |

**Highlights:**
- Crafted 4 behavioral features (utilization ratio, payment ratio) improving baseline by 8%
- Enabled targeted interventions for high-risk customer segments

---

### 2. 🏦 Loan Approval Prediction System
**File:** `Loan_Approval_Prediction.ipynb`

Built an automated loan decision system on 1,000 applications with a three-tier approval framework enabling 80% automation.

| Metric | Value |
|---|---|
| AUC Score | **0.803** |
| Precision | **87%** |
| Models Used | Logistic Regression, XGBoost, Decision Tree |
| Features Engineered | 8 custom risk-scoring features |

**Highlights:**
- Implemented cost-sensitive evaluation for financial risk optimization
- Achieved 87% better financial outcomes than tree-based alternatives
